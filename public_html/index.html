Educatec

En este proyecto vamos a mostrar que simple es crear un frontend web temático con la api de recursos educativos de educ.ar http://datosabiertos.educ.ar

Es ideal para aquellos que nunca hayan trabajado con web services rest y quieran aprender a hacerlo como para aquellos que quieran dar sus primeros pasos en la programación web en general.

Utilizamos Mustache, Jquery y Bootstrap

Documentación

Crear un elemento que busque recursos por un texto determinado

Atributos

  • data-buscar Establece la palabra a buscar, en caso de comenzar con "#" se interpretará que es un ID de un campo de texto
  • data-buscar-id Es un ID de un campo de texto del cual se tomará el valor del mismo para realizar la búsqueda
  • data-append Establece donde mostrar los resultados, serán agregados al final del elemento
  • data-html Establece donde mostrar los resultados, reemplazarán el html anterior
  • data-hide Establece que elemento ocultar al realizar la búsqueda
  • data-show Establece que elemento mostrar al realizar la búsqueda
  • data-source Establece que métodos de API utilizar (Juegos, Videos, Ebooks, Secuencias, Infografias), en caso de ser más de uno (separar por comas) y para buscar en todos simplemente no ponerlo

Ejemplos

BUSCAR MEDIANTE CLICK
Este elemento buscará recursos relacionados con robótica y los sumará a los que se encuentren dentro del elemento de ID "busqueda-resultados" <a data-buscar="robotica" data-source="Videos" data-html="busqueda-resultados" data-hide="descripcion-sitio" data-show="contenedor-resultados">Robótica</a> Robótica
BUSCAR MEDIANTE TEXTO
Este elemento buscará recursos relacionados con el valor del campo de texto de ID "texto-busqueda" y reemplazara el html del elemento de ID "busqueda-resultados" con los resultados <button> data-buscar-id="texto-busqueda" data-html="busqueda-resultados" type="submit" class="btn btn-success">Buscar</button>
